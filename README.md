# Sistema de Visualización: Tabla Periódica Interactiva
**Universidad Nacional de La Matanza (UNLaM)**  
**Cátedra:** Desarrollo Web  
**Año:** 2026

## 1. Descripción del Proyecto
Este repositorio contiene el desarrollo de una aplicación web técnica diseñada para la visualización y consulta de datos químicos. El objetivo principal es proporcionar una interfaz modular, eficiente y responsiva utilizando estándares modernos de desarrollo Web Vanilla.

## 2. Tecnologías y Estándares
*   **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
*   **Arquitectura de Estilos:** CSS Modular (Separación por capas: global, componentes y páginas).
*   **Persistencia:** LocalStorage API para la gestión de usuarios y favoritos.
*   **Metodología Git:** GitFlow simplificado mediante ramas de funcionalidades (feature branches).

## 3. Estructura del Repositorio
El proyecto sigue una estructura de directorios modular para facilitar el mantenimiento:

- `assets/`: Recursos estáticos (fonts, media).
- `css/`: Hoja de estilos jerárquica.
- `js/`: Lógica de negocio y manipulación del DOM.
- `index.html`: Punto de entrada principal de la aplicación.

## 5. Protocolo de Contribución
Para mantener la integridad de la rama principal (`main`), se requiere:
1. Sincronizar repositorio local: `git pull origin main`.
2. Trabajar exclusivamente en la rama de funcionalidad asignada: `feature/[nombre-interfaz]`.
3. Documentar los cambios mediante mensajes de commit descriptivos.
